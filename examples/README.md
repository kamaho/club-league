# Eksempler / Examples

Denne mappen inneholder visuelle referanser og eksempler som ikke er i bruk i appen, men som kan brukes som inspirasjon senere.

This folder contains visual references and examples that are not currently used in the app but can be used as inspiration later.

## tennis-court-card-visual.html

Eksempel på en tennisbane-inspirert kortdesign for å vise info visuelt:

- **Layout:** Baneform med header (øvre banehalvdel), nettbånd, service-bokser (hovedinfo), og footer (nedre banehalvdel).
- **Stil:** Clay-farger, kritt-hvite linjer, mineraltekstur, Space Grotesk / Space Mono.
- **Interaktivitet:** Hover med perspektiv-rotasjon (3D-effekt).
- **Animasjon:** Reveal-animasjon med forsinkelse på seksjoner.

Åpne filen i en nettleser for å se eksempelet.
